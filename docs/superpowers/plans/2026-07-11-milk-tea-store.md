# 三平方奶茶店实施计划

日期：2026-07-11  
状态：执行中

## 目标

将现有奶茶店顾客端和管理端改造成可在浏览器直接使用的中文应用，并保留现有 Flask 接口、SQLite 数据和订单闭环。

## 实施步骤

1. 先更新顾客端和管理端测试，覆盖中文界面、商品定制、购物车、下单、查询、登录、订单流转和商品管理。
2. 重构顾客端信息结构，完成中文品牌区、分类菜单、商品定制、购物车数量调整、订单结果和状态时间线。
3. 重构管理端，完成中文登录、经营数据、订单状态操作、商品新增编辑及启停。
4. 新增奶茶店独立样式文件，完成桌面与手机布局，并保持与工具中心一致的圆润、清晰和轻量视觉语言。
5. 在真实本地接口上完成顾客下单、订单查询和管理端处理流程。
6. 执行全部前端与后端测试、生产构建、差异检查和敏感内容检查。

## 验收标准

- 公开页面和管理页面的可见操作文案均为中文。
- 顾客可以完成筛选、定制、加入购物车、调整数量、下单和查询。
- 管理员可以登录、查看经营数据、更新订单状态并管理商品。
- 接口失败时购物车不丢失，按钮在提交期间不可重复点击。
- 桌面端和手机端没有横向溢出、文字裁切或控件重叠。
- 自动测试、生产构建和真实浏览器验收全部通过。
